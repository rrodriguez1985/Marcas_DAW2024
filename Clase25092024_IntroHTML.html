Introducción a HTML


Definición de Página Web

Una página es un documento que está formado por etiquetas o tag (marcas), que nos va a permitir mostrar cualquier
tipo de información: texto,imágenes, audio, vídeo...
Están formadas por etiquetas que están predefinidas: NO nos podemos inventar nombres
Para poderlas visualizar, sólamente necesitaremos un navegador web: IE, Edge, Firefox, Chrome, Opera...
Para crear páginas web, necesitaremos un Editor: VS Code, Sublime Text, Notepad ++, Atom, XCode... e incluso
podemos hacerlo con un block de notas!!!

Lenguaje de etiquetado HTML

El lenguaje de etiquetado HTML, viene del contexto HyperText Markup Language, Lenguaje de Marcado de HyperTexto
Utilizaremos etiquetas para insertar información de cualquier tipo y que además, nos permitirá insertar
lo que se conoce como hiperenlaces o hipertexto.
HTML, NO ES UN LENGUAJE DE PROGRAMACIÓN. ES UN LENGUAJE DE MARCAS
Estas etiquetas vienen predefinidas por W3Consortium (World Wide Web)
Permite aglutinar cualquier tipo de información que se desee.

Funcionamiento y visualización de un web

Etiquetas o Tag en HTML

  • Las marcas o tags en HTML, se van a escribir entre desigualdades (es decir, simbolo "<", y el símbolo ">")
  • Van a tener una etiqueta de apertura, y otra de cierre. Es decir, una de inicio y otro de fin
  • La etiqueta de inicio contendrá el nombre de la etiqueta, y el cierre,tendrá el nombre y delante del mismo /


Aunque todas las etiquetas por lo general tienen una de inicio y otra de fin, existen en html lo que se conocen
como monoetiquetas.Qué sólo tiene una etiqueta de apertura/inicio.
Eje: img, hr, br, etc.

Atributos en HTML

En HTML, tenemos lo que se conoce como atributos.Sirven para dar información o cualidades adicionales a nuestras etiquetas.
Todas las etiquetas pueden tener uno o varios atributos, y siempre se colocarán en la etiqueta de inicio.
"<"etiqueta atributo= valordeatributo atributo=valor de atributo ">"

Comentarios en HTML

Un comentario, sirve para dar información al programador o desarrollador de la web.
Dicha información, NUNCA se va a mostrar cuando ejecutemos la página web.
Para poder poner un comentario, lo haremos de la siguiente manera:
Se pone "<""!" seguido de "--" a continuación el texto a indicar y se cierra con "--"">"

Estructura de una Página Web en HTML

Los documentos en HTML, están divididos en dos partes:
  • Cabecera: donde se almacena información tal como el título de la pestaña del navegador,
    el idioma, links a librerías de JS(JavaScript), links a hojas de estilos externos (CSS),
    favicom, metadatos (UTF-8). Este contenido no se muestra en la web.
  • Cuerpo: incluye TODOS los elementos de nuestra página web: párrafos, texto, imágenes,
    vídeos, audios, formularios, etc.
